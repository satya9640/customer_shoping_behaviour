Customer Shopping Behavior Analysis
Project Overview

This project analyzes customer shopping behavior using SQL queries to extract meaningful business insights from retail transaction data. The goal of the project is to understand customer purchasing patterns, identify best-selling products, and support data-driven decision making for retail businesses.

Dataset

The dataset contains information related to:

Customers

Products

Orders

Order Items

Sales transactions

These tables simulate a retail database system used in e-commerce or supermarket businesses.

Objectives

The main objectives of this project are:

Analyze customer purchasing behavior

Identify best-selling products

Understand order patterns

Generate business insights using SQL queries

Practice real-world retail data analysis

Technologies Used

SQL

Relational Database Concepts

Data Analysis Techniques

Database Tables

The project uses the following tables:

customers – customer information

products – product details

orders – order transactions

order_items – product details within each order

Key Analysis Performed

Some of the important SQL queries included in this project:

Retrieve all customer data

Retrieve all product information

Retrieve order and order item details

Identify best selling products

Analyze customer purchase frequency

Calculate total sales per product

Generate business insights from transaction data
